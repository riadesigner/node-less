# авторизация с помощью passport

## библиотеки

- ejs
- express
- express-session
- passport
- passport-local

### роуты

GET /                 - главная  
GET /api/user/admin   - админка  
GET /api/user/login   - страница с формой входа / регистрации  
GET /api/user/me      - страница профиля  

POST /api/user/login  
POST /api/user/signup
